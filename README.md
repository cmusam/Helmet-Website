# Helmet-Website
Fast parallel primitives for Swift 2.0 using Metal API

This is my final project for 15-418 Parallel Computer Architecture and Programming in Spring 2016. By using [Metal](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&uact=8&ved=2ahUKEwi7sdXugbrfAhVKjlQKHSTQBfkQFjAAegQIChAB&url=https%3A%2F%2Fdeveloper.apple.com%2Fmetal%2F&usg=AOvVaw1sF6QKyL9j6rbP4tSbpTVG), 
this simple framework hides underlying GPU accesses and instead presents users with parallelized and ultra fast primitives including `scan`, `filter`, `map` functions.
